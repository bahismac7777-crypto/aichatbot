# Project Dependencies

To install all dependencies for this project, run the following command in your terminal:

```bash
npm install
```

## Production Dependencies
These are the core libraries used to build the chatbot:

- **@google/genai**: Google's official SDK for the Gemini AI models.
- **react**: The core UI library.
- **react-dom**: React's entry point for the browser.
- **react-router-dom**: For page navigation (Home and Chat pages).
- **lucide-react**: Beautiful icons used throughout the interface.
- **motion**: For smooth animations and transitions.
- **react-markdown**: To render the AI's responses with proper formatting.
- **vite**: The build tool and development server.
- **tailwindcss**: For styling the application.
- **clsx** & **tailwind-merge**: Utilities for managing CSS classes.
- **dotenv**: For managing environment variables (like your API key).
- **express**: Backend server framework.

## Development Dependencies
These are tools used during development:

- **typescript**: For type-safe coding.
- **tsx**: To run TypeScript files directly.
- **@types/node** & **@types/express**: Type definitions for Node.js and Express.
- **autoprefixer**: For cross-browser CSS support.

---
*Note: All these are already listed in your `package.json` file.*
